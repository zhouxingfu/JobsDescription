除此之外，还有很多公司提供了比较细节的要求，比如旷视、今日头条、九坤投资和建信金科。

从上面看，要想入门HPC，要具备以下条件  
1. 语言基础
   C++ python
2. 计算机体系结构
   CSAPP，比如CPU架构，这里面如果深入的话，可以学习memory hierarchy，memory access pattern
   汇编  
   指令级编程 x86 AVX SSE、arm neon
3. 并行编程
   OpenMP MPI 
4. MKL BLAS cuDNN是矩阵计算库
